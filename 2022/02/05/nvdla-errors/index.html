6.编译报错util/memfd.c:44:12: error: static declaration of ‘memfd_create’ follows non-static declara

解决方法：修改报错的文件memfd.c

按照图中所示的报错信息，找到memfd.c文件的第40行，将“static”声明修改为“int”

static int memfd_create(const char *name, unsigned int flags)

int memfd_create(const char *name, unsigned int flags)

修改后保存退出，make，然后make clean。
